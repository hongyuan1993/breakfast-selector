# 🍳 大宝早餐选择

温馨的橙黄色渐变风格早餐选择应用，支持登陆、修改、选择早餐及查看历史记录。

## 技术栈

- React 18 + Vite 5
- Tailwind CSS 3
- localStorage 数据存储

## 功能

- **登陆早餐**：从预设中选择要加入早餐库的选项，自动分类
- **修改早餐**：修改已登陆早餐名称，或删除
- **选择早餐**：多选今日早餐，支持选项（如煎鸡蛋选全熟/半熟/流心）
- **历史记录**：以日历形式查看已保存的早餐记录

## 运行

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
npm run preview
```

## 预设早餐

小锅米线、蒸玉米、煮鸡蛋、煎鸡蛋、三明治、厚蛋烧、煮燕麦片、蒸包子、草莓、香蕉、烫饭、黑米红豆榨汁
